<h2>Navbar CSS</h2>
<p>Demo of dynamic CSS styling with animations and themes.</p>

<p><img src="./Screenshot 2022-01-17 at 16-38-31 Navbar CSS.png" /></p>

<h3>Description</h3>
<p>Navbar CSS is a demo of frontend styling that contains beautiful animations, responsive design, dynamic navbar and day-time / night-time themes.</p>

<h3>Technical description</h3>
<p>All animations and responsiveness is handled by CSS. Icons are dual-color SVGs. Main content is formed with Grid styling. Changing themes function is done by Javascript code that changes body class and saves it in local memory.</p>

<h3>Installation</h3>
<p>Just open with browser.</p>
